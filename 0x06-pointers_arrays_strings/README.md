# 0x06-pointers_arrays_strings

This project also focuses on learning more about pinters, strings and arrays in C programming language.

## Files

Below are the files worked on in this project.

| File name | Description |
| --------- | ----------- |
| 0-strcat.c | concatenates two strings |
| 1-strncat.c | also concatenates two strings |
| 2-strncpy.c | this function copies a string |
| 3-strcmp.c | this file compares two strings |
| 4-rev_array.c | a function that reverses the content of an array of integers |
| 5-string_toupper.c | this function, changes all lowercase letters of a string to uppercase |

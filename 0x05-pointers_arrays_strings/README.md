# 0x05-pointers_arrays_strings

Project done during ALX software engineering training. This project focuses on learnjng how to use pointers with arrays and strings in C.

## Files

Below are all the files created in this project and their descriptions.

| Filename | Description |
| -------- | ----------- |
| `0-reset_to_98.c` | takes a pointer to an int as parameter and updates the value it points to to 98. |
| `1-swap.c` | returns the length of a string. |
| `3-puts.c` | prints a string, followed by a new line, to stdout. |
| `4-print_rev.c` |  prints a string, in reverse, followed by a new line |
